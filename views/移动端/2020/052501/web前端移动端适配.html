            # 移动端适配

            # 一. 物理像素和设备独立像素

            设备 尺寸（寸） 物理像素（pt） 设备分辨率(px) 缩放因子 ppi
            3GS 3.5 320*480 320*480 1.0 163
            5(s) 4.0 320*568 640*1136 2.0 326
            6(s)/7/8 4.7 375*667 750*1334 2.0 326
            x(s) 5.8 357*812 1125*2436 3.0 458

            • 相关单位

              • pt逻辑分辨率（视觉单位，和屏幕尺寸有关系）
              • px物理分辨率（像素点）
              • ppi像素（每英寸包含的像素点）

            • 单位换算关系

              • px是pt的二倍 （以iphone6为例）

            # 二. vw与vh

            vh: 相对于视窗的高度, 视窗被均分为100单位的vh （1vh 视窗高度的百分之一） vw: 相对于视窗的宽度, 视窗被均分为100单位的vw（1vw 视窗宽度的百分之一）

            # 三. rem

            rem 相对长度单位。相对于根元素(即html元素) font-size计算值的倍数，其默认为16px

            # 四.vscode安装Easy LESS

            由于样式部分使用less写的，需要安装Easy LESS插件

            从vscode商店中下载Easy LESS插件，安装即可

            # 五. 移动端适配具体实现

            首先我们以iphone6 为准把设计图的宽度设置成750px（iphone6设备的设备分辨率为750px）方便我们后续进行计算。

            其次，我们把根元素(即html元素)设置成font-size:10vw（75px），即是把font-size设置成屏幕宽度的十分之一。

            然后，我们把.box的宽度设置成5rem，既是把.box的宽度设置成屏幕宽度一半（375px）。

            接下来，根据这思路来编写代码看看效果是否是正确的
